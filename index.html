Sou formado em Administração de Empresas, com MBA em Gestão Financeira e Controladoria, além de formação em Análise e Desenvolvimento de Sistemas. Minha trajetória profissional me proporcionou uma visão estratégica abrangente, permitindo integrar conhecimentos financeiros e gerenciais com habilidades técnicas voltadas para a criação de soluções digitais.

A experiência como administrador de empresas foi fundamental para compreender as necessidades de diferentes modelos de negócios, o que me capacita a desenvolver sites, landing pages e sistemas personalizados que realmente atendem aos objetivos estratégicos de cada cliente.

Com foco em inovação e eficiência, estou sempre em busca de soluções que agreguem valor, seja para pequenas empresas em crescimento, startups ou negócios já consolidados no mercado.

Alguns Projetos

Landing Page

Este modelo de Landing Page apresenta os produtos e beneficios de uma barbearia fictícia, sua localização, vídeo marketing e formulário de contato para atendimento.

Cardápio Digital

Este é um cardápio digital onde são apresentados os produtos e preços de um estabelecimento comercial. O usuário administrador poderá criar, editar e deletar os produtos. Um documento em PDF com a lista dos produtos também poderá ser baixado pelo usuário administrador.

Portfolio

Neste projeto foi desenvolvido um portfolio básico personalizado de acordo com as necessidades da cliente. Foi utilizado HTML, CSS e JavaScript. A ideia foi criar um site que apresentasse os trabalhos realizados, serviços prestados e informações sobre o cliente. Formulários de contato com integração ao Discord também foram desenvolvidos.

Gestão Financeira Pessoal

Sistema de gestão financeira pessoal, com controle de despesas, receitas e balanço financeiro desenvolvido com framework Laravel. Dashboards com gráficos e relatórios detalhados.

Sistemas Personalizados

Sistemas desenvolvidos com framework Laravel, bootstrap e javascript para controle de estoque, inventário de dados, automação de processos, contratos e muito mais. Recursos completos para o desenvolvimento de soluções empresariais, com foco em eficiência e qualidade. *Imagem ilustrativa do projeto.

Calendário Dinâmico

Este calendário foi desenvolvido com framework Laravel, bootstrap e javascript para gerenciamento de eventos. Ele oferece uma interface simples e intuitiva para adicionar, editar e excluir eventos com facilidade.
